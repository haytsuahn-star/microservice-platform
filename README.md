# Microservice Platform (Docker + Kubernetes)

## Описание

Проект представляет собой распределенную систему на Go с использованием Docker и Kubernetes.

## Архитектура

- Go backend (REST API)
- Docker контейнеризация
- Docker Compose для локального запуска
- Kubernetes Deployment + Service
- PostgreSQL (в docker-compose)
- Healthcheck endpoint (/health)

## Технологии

- Go
- Docker
- Docker Compose
- Kubernetes
- GitHub Actions
